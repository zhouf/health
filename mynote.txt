在windows系统下把默认编辑器配置成notepad
git config --global core.editor C:/windows/notepad.exe

撤销对file2.jsp的编辑，恢复到上次提交状态，注意两个-- 后面有空格
git checkout -- file2.jsp

查看最近再次的提交记录
git log -2

将文件进行移动
git mv adduser.jsp users/add.jsp

添加所有文件到暂存区
git add .

提交暂存区文件内容
git commit -m "some comments"

查看远程仓库
git remote show

查看远程仓库信息
git remote -v
